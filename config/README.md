# DAO Configuration

This directory contains configuration files for the DAO.

## Files

- **governance-params.json**: Governance parameters for the DAO
- **network-config.json**: Network-specific configurations

## Governance Parameters

You can modify these parameters before deployment:

- **votingDelay**: Number of blocks before voting starts
- **votingPeriod**: Duration of voting in blocks
- **proposalThreshold**: Minimum tokens needed to create a proposal
- **quorumPercentage**: Percentage of votes needed for quorum
- **timelockDelay**: Delay before executing passed proposals (in seconds)
